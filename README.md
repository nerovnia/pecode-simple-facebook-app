# Simple facebook like application

**Please use:**

- Node.js: `>=20`
- database: `PostgreSQL`

Create .env

```text
APP_PORT=port
DATABASE_HOST=host
DATABASE_PORT=port
DATABASE_DBNAME=dbname
DATABASE_USER=myuser
DATABASE_PASSWORD=mypassword
```

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```
